\section*{Abstract}
\vspace{-10pt}
The objective of this thesis was to make a functional application for robot control and mapping. Mapping is done in parallel with the robot control. A 2D map is being made which is shown under the robot. Laser scan data is displayed in a form of spheres and a first person view from the robot camera was also enabled. For research purposes in the case of two robots in the same simulation, a feature was created in order to switch the camera view from one robot to another. 3D mapping is also implemented, but on a separate scene. The application is functional on three platforms: Windows, Linux and Android.
